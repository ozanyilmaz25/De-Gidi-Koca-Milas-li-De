# De-Gidi-Koca-Milas-li-De
ne kadar Milas'SIN testi

Muğla'nın Milas ilçesine bağlı köyleri (Mahaleler-Büyükşehir Yasası) bilip bilinmediğini test etmeyi amaçlayan HTML / JavaScript tabanlı internet tabanlı bir coğrafya oyunudur. Oyuncular, verilen süre içinde kendilerinden istenen Milas köylerini harita üzerinde sınır, konum ve Esri uydu görüntüsü yardımıyla bilmeye çalışacaktır.
